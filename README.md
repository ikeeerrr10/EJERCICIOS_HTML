# EJERCICIOS_HTML
Mis primeros ejercicios de HTML sencillos, para el módulo LMSG :grinning:

## Descripción
En este repositorio de GitHub vamos a ir subiendo ejemplos de páginas web realizadas para el módulo LMGSI.
Lógicamente, las primeras págins serán muy sencillas y, poco a poco, irán creciendo en complejidad.

## Entorno de desarrollo integrado 
Para crear y editar las páginas utilizamos el IDE ligero **Visual Studio Code**.
Para enviar las páginas al explorador, utilizamos la extensión **Live Server** que hemos instalado en VS Code.
Para *intrepretar* las páginas, en principio, nos sirve cualquier exlorador (por ejemplo, Chrome o Firefox).

[Página oficial de VS Code](https://code.visualstudio.com/)
